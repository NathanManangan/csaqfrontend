Introduction

Binary Search is a searching algorithm used in a sorted array by repeatedly dividing the search interval in half. This topic can be tested on the AP Exam but it isnt commonly seen.



Speed

- Binary search is one of the fastest searching algorithms - When compared to a linear search, it is much faster the bigger the array is


- Linear search is O(n) - Whereas binary search is O(log n)



More of how it works

- Set the low index to the first element of the array and the high index to the last element

- Next, set the middle index by averaging the low and high index - this will look like (high+low)/2

- If the element at the middle index is the target element, return the middle index

- If not begin the search by creating the two halves of the index

- If the number is less than the element at the middle index, set the high index to middle index – 1

- If the number is greater than the element at the middle index, set the low index to middle index + 1

- Repeat the previous steps until the wanted number is found
